## Risks & Mitigations

### Risk 1: Complex Terraform Interdependencies
- **Impact**: Fixing terraform duplication might break other configs
- **Mitigation**: Test each provider block change independently
- **Rollback**: git checkout original version if needed

### Risk 2: Newly Generated Lock Files Diverge
- **Impact**: npm lock files generated in CI environment might differ locally
- **Mitigation**: Generate lock files on identical Node version as CI (v20.20.2)
- **Verification**: checksum matching or strict mode

### Risk 3: Linting Failures Uncovered
- **Impact**: Removing lint-report.txt check might hide actual lint errors
- **Mitigation**: Run `npm run lint` locally to verify
- **Alternative**: Keep lint report but generate it in CI
